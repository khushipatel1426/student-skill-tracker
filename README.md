# 📊 Student Skill Progress Tracker

## About this project

This project helps students track their skill progress in a simple way. Users can add skills and monitor their improvement over time.

---

## What I did

* Created a simple UI using HTML and CSS
* Used JavaScript to handle user input and updates
* Added feature to track and display progress
* Designed layout to make it easy to use

---

## Technologies used

* HTML
* CSS
* JavaScript

---

## How to run

1. Download or clone the project
2. Open the folder
3. Run `index.html` in your browser

---

## Files in this project

* `index.html` → main structure
* `style.css` → styling
* `script.js` → logic

---

## Output

* Users can add and track skills
* Displays progress clearly
* Simple and interactive interface
